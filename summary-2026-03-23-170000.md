## Key AI Topics Discussed

- **"Ralph" Loops / AFK Agents** — Matt Pocock's workflow for autonomous Claude Code agents that work in the background while you're away. Uses GitHub issues as tasks, iterates through a PRD, and runs in sandboxed mode. Members actively sharing their implementations and variations.

- **Claude Code Workflow & Skills** — Extensive discussion on structuring Claude Code with custom skills (`/issue-triage`, `/step`), CLAUDE.md vs AGENTS.md configurations, and using commits (not progress.txt) to retain context between sessions.

- **AI SDK v6 (Vercel)** — Migration from v5 to v6, breaking changes with Evalite compatibility, OpenRouter as alternative provider, and structured output generation.

- **Evalite (AI Eval Framework)** — Matt's TypeScript evaluation framework for testing LLM outputs. Discussions on variant comparison (A/B testing prompts), conditional scorers, filtering datasets, and rate limit management.

- **Voice-to-Text for Coding** — Comparisons of Wispr Flow, Aqua Voice, and VoiceInk for dictating to Claude Code. Wispr Flow praised for understanding code terminology and file paths.

- **Sandboxing Claude Code on Mac** — Built-in macOS sandboxing eliminates need for Docker. Setup via `.claude/settings.json` with sandbox enabled.

- **Multi-Agent Orchestration** — Discussion on orchestrator/router patterns, human-in-the-loop design, and using Durable Agents via Workflow DevKit.

- **Team Collaboration with AI Agents** — How 5+ engineers collaborate when no one reviews AI-generated code. Tension between traditional code review and AI-driven quality assurance via tests/interfaces.

## Key Takeaways

1. **Ralph loops are the hot pattern** — Autonomous background agents that pick up GitHub issues, implement them, and iterate. Worth setting up for your projects.
2. **Voice dictation is a game-changer** — Multiple members report significant productivity gains. Try Wispr Flow or Aqua Voice.
3. **Evalite needs v6 update** — If you're on AI SDK v6, Evalite compatibility is still being worked on.
4. **Sandboxing on Mac is free** — No Docker needed, just configure `.claude/settings.json`.
5. **Upcoming cohort starts March 30** — "Claude Code for Real Engineers" covers requirements gathering to AFK agents.
6. **Context management matters** — Clear context between tasks. Use commits as context carriers, not progress files.
7. **Figma MCP + pixel-perfect skills** dramatically reduces design-to-code iterations.
